# AirCargo SQL Analysis

A MySQL project analyzing airline customer, ticket, and route data for Air Cargo.

## Project Goals
- Identify frequent customers
- Analyze route and class-wise travel trends
- Calculate revenue and ticket metrics
- Improve query performance

## Files in This Repository
- AirCargo.sql — All SQL queries written for the assignment
- Report.pdf — Project summary and conclusions
- /screenshots — ER diagram, query outputs, analysis visuals

## SQL Concepts Demonstrated
- Joins, subqueries and aggregations
- IF() conditional statements
- Window functions
- Views and stored procedures
- Indexing + EXPLAIN plan
- GROUP BY + WITH ROLLUP

## Key Learnings
This project strengthened my skills in:
- Database creation and relational modeling
- Writing clean and optimized SQL
- Extracting insights from real-world datasets
